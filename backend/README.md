# Backend

백엔드 영역입니다.

## 역할

- 인증
- 강사용 API
- 학생 앱 API
- PostgreSQL 연동
- 성적 계산 및 제출 처리

## 예정 기술 스택

- Node.js
- NestJS 또는 Express
- TypeScript
- PostgreSQL

## 작업 규칙

- `backend` 작업 중에는 `admin-web` 또는 `student-app`을 수정하지 않습니다.
- API 구조는 `docs/API.md`를 기준으로 합니다.
- DB 구조는 `docs/ERD.md`를 기준으로 합니다.
- 환경 변수 예시는 루트 `.env.example`을 기준으로 합니다.

## 현재 상태

협업을 위한 폴더만 생성되어 있으며 기능 구현은 없습니다.
