# Student App

학생용 앱 영역입니다.

## 역할

- 학생 로그인
- 배포된 문제집 목록 조회
- 문제 풀이
- 제출 결과 확인
- 내 성적 확인

## 예정 기술 스택

- React Native
- TypeScript

## 작업 규칙

- `student-app` 작업 중에는 `admin-web` 또는 `backend`를 수정하지 않습니다.
- API 구조는 `docs/API.md`를 기준으로 합니다.
- 공통 타입이 필요하면 `/packages/types`에 추가합니다.

## 현재 상태

협업을 위한 폴더만 생성되어 있으며 기능 구현은 없습니다.
